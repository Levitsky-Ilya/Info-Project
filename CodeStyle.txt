CodeStyle
=======

//Стандарты оформления кода, изменения предварительно обсуждаются

Отступы/табуляция	- 4 пробела

Константы			- большими буквами с нижним подчеркиванием
Типы				- смешанный регистр, начиная с заглавной буквы ("SavingsAccount")
Переменные			- смешанный регистр, начиная с нижнего ("savingsAccount")
Методы и функции	- смешанный регистр, начиная с глагола в нижнем ("getName()")

После if, for, while и т.д. ставятся пробелы, в скобках после них ставятся пробелы между знаками, а между скобками и внутренностями нет.
(Вообще, Visual сам за вас все решит)



ФИГУРНЫЕ СКОБКИ
Для функций и классов (и структур!)
	int main()
	{
		//some code
	}

Для if, while, for и т.п.
	if ( ... ) {
		//some code
	} else {
		//another code
	}



Коментарии на АНГЛИЙСКОМ ЯЗЫКЕ:
	Для прототипа функции	- после 
	Для класса				- до
	Для функции в классе	- перед реализацией в отдельном .cpp-файле
	
При описании класса и функции комментарии выделяются так
/*
 * Blah-Blah
 * Blaaah
 */

Короткие поясняющие комментарии - справа от текста, "//"
